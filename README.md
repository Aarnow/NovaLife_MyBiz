# Novalife Plugin

Ce plugin Novalife MyBiz permet de consulter le répertoire des sociétés depuis MyMenu

## Table des Matières

- [Installation](#installation)
- [Utilisation](#utilisation)
- [Droits de Propriété Intellectuelle](#droits-de-propriété-intellectuelle)

## Installation

1. Téléchargez le fichier `MyBiz.dll` depuis la page des releases de ce dépôt. 
2. Ajoutez le fichier `MyBiz.dll` dans le dossier des plugins de votre serveur Novalife.

## Utilisation

Vous devez posséder le plugin [MyMenu](https://github.com/Aarnow/NovaLife_MyMenu/releases) et être administrateur de votre serveur.
Activez le service admin et ouvrez MyMenu.
Vous y trouverez MyBiz, où vous pourrez cconsulter les sociétés de votre serveur.

Les propriétaires d'une société peuvent modifier la description, les informations de contact, ainsi que l'état du recrutement.

Si un joueur employé au sein d'une société est en service actif, la société est notifiée comme étant "ouverte" dans le répertoire.
